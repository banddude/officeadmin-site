Stack

Everything runs on a Mac Mini. No Kubernetes, no containers, no cloud compute. Just Python, SQLite, and tmux.

Languages

  • Python 3 — primary language for all CLIs and modules
  • Bash — agent orchestration, system scripts, CLI wrappers
  • Node.js — QuickBooks OAuth, some API integrations
  • JavaScript — n8n workflow expressions

AI and LLMs

  • Claude (Anthropic) — primary reasoning engine via Claude Code and API
  • MCP Servers — tool access layer connecting Claude to all modules
  • AssemblyAI — speech-to-text with speaker diarization
  • Pinecone — vector embeddings for speaker identification

Databases

  • SQLite — MemPalace knowledge graph, conversations, contacts, messages
  • GitHub — entity CRM (JSON + Markdown files in a repo)
  • Apple Notes — AIVA's soul/personality source (iCloud-synced)

Apple Ecosystem

  • EventKit — Calendar and Reminders access
  • PyObjC — Python bridge to macOS frameworks
  • iCloud — sync for Notes, Reminders, Calendar, voice memos
  • APNS — push notifications to all devices
  • Messages.app — chat.db access for iMessage search

Networking

  • Tailscale — mesh VPN connecting all devices
  • Cloudflare — DNS, Workers, edge routing, SSL
  • Cloudflare Tunnel — secure ingress without port forwarding
  • Railway — n8n workflow hosting

Business APIs

  • QuickBooks Online — accounting, estimates, invoices
  • Google Workspace — Gmail, Calendar, Drive, Sheets, Chat
  • QuickBooks Time — crew scheduling and timesheets
  • Alpaca Markets — stock and crypto trading

Communication

  • BlueBubbles — cross-platform iMessage relay
  • Home Assistant — smart home device control
  • Playwright — browser automation (MCP server)
  • Flask — notify API server for message routing

Infrastructure

  • Mac Mini — the one server that runs everything
  • tmux — background agent session management
  • launchd — macOS daemon supervision
  • macOS Keychain — credential storage for all OAuth tokens
  • GitHub Actions — entity CRM validators

Why a Mac Mini?

Most of what AIVA does requires access to macOS-native APIs. Contacts, Reminders, Calendar, Notes, and Messages all use Apple frameworks (EventKit, PyObjC, SQLite databases in ~/Library). iCloud sync means changes on the Mac Mini show up on Mike's phone instantly. The Apple ecosystem is the integration layer. A Linux server would need to replicate all of that from scratch.

The Mac Mini sits at home, always on, connected via Tailscale. Cloudflare Tunnels handle secure ingress. There is no cloud compute bill for the core system. The only hosted services are Railway (n8n) and the various third-party APIs.
