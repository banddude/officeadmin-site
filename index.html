What it actually does

Every operational task that used to happen by hand now either runs automatically or gets drafted for review.

Email triage

Watches Gmail, classifies messages as action items, FYI, or noise. Drafts replies in Mike's voice and queues them for approval before sending.

Estimates and invoices

Programmatic access to QuickBooks Online. Creates estimates, invoices, bills, and customer records without opening the app.

Contact management

Direct read/write to Apple Contacts via SQLite. Every customer, vendor, and crew member is an entity with linked tasks, work records, and communication history.

Voice memos

Transcribes recordings with AssemblyAI and identifies speakers using Pinecone voice embeddings. Meeting notes turn into tasks and entity updates automatically.

Scheduling and crews

Integrates with QuickBooks Time for crew scheduling and timesheets. Calendar events sync across Apple Calendar and Google Calendar.

Knowledge graph

MemPalace is a SQLite knowledge graph with thousands of entities and relationships, extracted from conversations, documents, and agent sessions.

How it works

A Mac Mini at home runs the whole system. Everything else connects to it.

The brain

A Mac Mini running Python CLIs, n8n workflows, tmux-managed agent sessions, MCP servers, and Claude model calls. It is always on and always connected.

The reach

Phones, laptops, and speakers throughout the house are all participants. Everything syncs via Tailscale VPN and iCloud. AIVA can talk to Mike through iMessage, voice, or push notifications.

The memory

A GitHub repo serves as both an operational workspace and a lightweight CRM. Each person and company gets an entity folder with profile data, tasks, and work records.

Meet AIVA

AIVA is the AI agent that runs the system. It is not a chatbot and not a SaaS product. It is a personal operations system that Mike built for himself.

AIVA manages background agent sessions in tmux, runs scheduled jobs, responds to iMessages via BlueBubbles, drafts emails in Mike's voice using a learned rule set, and coordinates across 50+ modules. It can speak out loud through connected devices, show interactive prompts on Mike's phone, browse the web with Playwright, and manage smart home devices through Home Assistant. When it writes something that needs approval, it creates an Apple Reminder. When it learns something new, it stores it in MemPalace. The system is real, running in production, and handles the daily operations of a construction business.

Built by Mike Shaffer. Powered by Claude, Python, and a Mac Mini.
